# JWTTokenUtility

A lightweight and secure utility library for generating, verifying, and decoding JWT tokens in Node.js using TypeScript.

## Features

- Create JWT tokens with custom or default expiration times.
- Verify JWT tokens for authenticity and expiry.
- Decode JWT tokens to retrieve the payload.
- Handle token expiration validation.
- Easy-to-use API with TypeScript support.

## Installation

To install the library, you can use npm or yarn:

```bash
npm install jwt-token-utility
# or
yarn add jwt-token-utility
